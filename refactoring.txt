Весь JavaScript код связанный с админ-панелью перенести в /bod/bod.js;
Весь JavaScript код связанный с редактором и ckeditor перенести в /bod/editor.js;
Весь остальной PHP-код из файла crud.php перенести в файл db-api.php;
2. Файл index.php оставляем без изменений.
3. Файл style.css разделить:
Весь CSS относящийся к админ панели, включая стили редактора,

Файл app.js переместить в каталог /assets.
